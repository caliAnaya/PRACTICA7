FRONTEND

Llamamos frontend a la parte del sitio web con la que el usuario interactúa. Es lo que algunos suelen llamar “el lado del cliente. ... En otras palabras: el desarrollador frontend coge la propuesta del diseñador gráfico, y la convierte en realidad a través de diferentes lenguajes de programación.

En el pasado, los sitios web eran sitios de texto simple y estático con un poco de formato y tal vez incluso algo de animación. Todo eso fue gracias a HTML y CSS. Es importante tener en cuenta que el desarrollo de front-end ha cambiado significativamente en los últimos 10 a 15 años con el crecimiento explosivo de JavaScript, que no era tan ubicuo en el front-end como lo es ahora, ni tan común en el back-end.


Algunas de las tecnologías centrales utilizadas en el desarrollo web front-end incluyen:


HTML: el organizador


HTML es la forma en que se organizan todos los sitios en la web, por lo que no puede vivir sin el desarrollo de front-end. HTML5 es la última especificación HTML. Obtenga un resumen completo de esta columna vertebral en evolución de la tecnología web front-end y conozca una de sus funciones más interesantes, Canvas. Lo que una vez fue una herramienta complementaria para hacer que las páginas web fueran más interactivas, ahora es la tecnología del lado del cliente más ubicua. JavaScript es más que un lenguaje: es un ecosistema completo que abarca marcos, ejecutores de tareas, desarrollo del lado del servidor y más.


CSS: el estilista

Hojas de estilo en cascada (CSS) es la forma en que los desarrolladores agregan estilo y efectos a un sitio web. Los estilos se pueden agregar globalmente y luego superponerse sin cambiar ese estilo fundamental que se aplica a todo un sitio. CSS siempre está evolucionando. Conozca las novedades con la última y mejor versión, CSS3. Muchos desarrolladores de aplicaciones para usuario utilizan un preprocesador de CSS para hacer que escribir muchas líneas de CSS sea aún más rápido. Los marcos de front-end CSS como Foundation o Bootstrap pueden ayudarlo a crear sitios web pulidos en un instante. Sería negligente hablar sobre el desarrollo front-end sin mencionar el aspecto del diseño, dos cosas que van de la mano en el desarrollo de sitios web y aplicaciones. Muchos desarrolladores de front-end usan algunos sombreros, con diseñadores web y habilidades de UI / UX en su caja de herramientas. Si un diseñador sabe qué tipo de cosas son posibles con el código front-end, estará en mejores condiciones de diseñar interfaces de usuario interactivas e intuitivas, y viceversa.
